# Google Sheets Clone

## Getting Started

### Prerequisites
- Node.js and npm installed

### Installation
1. Clone the repository:
   ```sh
   git clone <repository-url>
   cd google-sheets-clone
   ```
2. Install dependencies:
   ```sh
   npm install
   ```
### Running the App:
   ```sh
   npm run start
   ```
Open http://localhost:3000 in your browser.

### Tech Stack
1) React
2) Handsontable
3) TypeScript
4) Tailwind CSS
